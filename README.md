# System Check Tool 🖥️

A bash script that monitors server health on Linux systems.

## Features
- Displays server info (hostname, user, date, IP)
- Checks status of critical services (sshd, firewalld, crond)
- Shows OK or FAILED for each service

## How to Run
chmod +x system_check.sh
./system_check.sh

## Tech Stack
- Bash Scripting
- Linux (Rocky Linux 9)
- systemctl
